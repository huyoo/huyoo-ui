# huyoo ui

个人练手组件库，基于 react+antd+less开发，部分组件使用了 antd 的基础组件 [react-component](https://github.com/react-component)开发。
文档地址：[]()

## 组件
- 年份选择器
- 列表选择器(开发功能中)
- cron表达式构建器(待开发)
- 表格选择器(待开发)

## 版本依赖
```
react >= 16.9.0
antd >= 4.14.1
```

## 快速上手
```bash
# 创建目录
$ mkdir myapp && cd myapp

# 安装依赖
$ yarn add antd
$ yarn add huyoo-ui
```

## 本地启动
```bash
# 克隆项目
$ git clone huyoo/huyoo-ui

# 安装依赖
$ yarn -i

# 组件打包
$ yarn run build:npm

# 启动本地文档(由于 demo 中有组件引入，需要先执行上一步)
$ yarn run dumi
```
