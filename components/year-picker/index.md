# YearPicker 年度选择器
## 属性
| 参数 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| className | 选择器 className | string | - |
| defaultValue | 默认年度 | string | - |
| disabled | 禁用组件 | boolean | false |
| size | 选择框大小 | `large` \| `middle` \| `small` | small |
| style | 自定义输入框样式 | CSSProperties | - |
| value | 年度 | string | - |
| onChange | 选择数据时的回调 | function(value:string) | - |
